## V · 术语精化声明 · IMPORTANT Terminology Notice

**2026.05.19 R6 后,以下术语精化生效**:

| 旧名(已撤回) | 状态 | 新名 |
|---|---|---|
| 美学直觉 | 矛盾术语 · V0.2 文档作为档案保留;新文档不再使用 | 艺术美直觉(对作品)+ 自然美直觉(对自然) · 或合称 **审美直觉** |

**理由**:"美学" = 关于美的学问;"美"在 TMA II 公理中只能形容人;人对人是共振不是直觉。
所以"美学直觉"将"美(人事件)"与"直觉(非人对象判断)"绑在一起 = 矛盾术语。
新文档(V0.3+)使用精化术语。V0.2 / TMA V1.3 等旧文档保留原貌作为档案。
